## Problem Statement

Ke mana orang Indonesia bergerak, dan mengapa?

Data migrasi resmi BPS SP2020 menunjukkan pola yang tidak sederhana:
DKI Jakarta kehilangan 585.011 penduduk bersih, sementara Jawa Tengah
justru menjadi penerima terbesar. Analisis ini memetakan arus tersebut
secara spasial dan mencari korelasinya dengan faktor ekonomi.
